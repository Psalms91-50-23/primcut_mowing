# Happy Property Frontend

Frontend application for Happy Property — a property maintenance and management platform built with Next.js and TypeScript.

This project provides the client-facing and employee management interface for handling:
- Quote management
- Property maintenance job management
- Employee and client interactions
- Image uploads and galleries
- Authentication and role-based access
- Automated email notifications

## Tech Stack

- Next.js
- TypeScript
- React
- Supabase
- Tailwind CSS
- Railway
- Resend
- Google Workspace

## Features

- Secure authentication and protected routes
- Quote creation and approval workflow
- GST, subtotal, and total calculations
- Image upload with modal preview support
- Employee dashboard and job tracking
- Responsive UI for desktop and mobile
- Toast notifications and error handling
- Real-time database integration with Supabase
